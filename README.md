# Smart Retail Analytics & Prediction System

## Overview
This project is an **end-to-end Machine Learning–driven analytics platform** for an e-commerce retail business.  
It includes **data cleaning, exploratory data analysis (EDA), customer segmentation, churn prediction, sales forecasting**, and deployment as an **interactive Streamlit web application**.

The project demonstrates a complete data science workflow from raw transactional data to actionable business insights.

---

## Features
- **Data Cleaning & Preprocessing**: Handles missing values, negative transactions, and feature engineering (`TotalPrice`).
- **Exploratory Data Analysis (EDA)**: Monthly sales trends, top products, top countries, customer purchase behavior.
- **Customer Segmentation**: RFM Analysis + KMeans clustering to identify loyal, at-risk, and occasional customers.
- **Churn Prediction**: Random Forest model predicts likelihood of customer churn.
- **Sales Forecasting**: Time series forecasting using ARIMA and Prophet.
- **Streamlit Web Application**: Interactive dashboard for KPIs, segmentation, churn prediction, and sales forecast.

---

## Dataset
- **Online Retail II dataset (UCI)**  
- Real-world e-commerce transactions (UK, 2009–2011)  
- 1,067,371 records with features:
  - Invoice, StockCode, Description, Quantity, InvoiceDate, Price, Customer ID, Country
